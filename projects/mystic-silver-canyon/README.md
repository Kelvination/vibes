# Quarto Game

A web-based implementation of the classic board game Quarto.

## About Quarto

Quarto is a two-player strategy board game where the goal is to get four pieces in a row that share at least one common attribute.

### Game Features

- **16 Unique Pieces**: Each piece has 4 binary attributes:
  - Height: Tall or Short
  - Color: Light or Dark
  - Shape: Square or Circle
  - Fill: Solid or Hollow

- **4x4 Game Board**: Place pieces strategically to create winning combinations

- **Unique Twist**: Your opponent chooses which piece you must play!

## How to Play

1. **Player 1's Turn**: Select a piece for Player 2 to place
2. **Player 2's Turn**: Place the selected piece on any empty space on the board, then select a piece for Player 1
3. **Winning**: Get four pieces in a row (horizontal, vertical, or diagonal) that share at least one common attribute
4. **Continue**: Alternate turns until someone wins or the board is full (draw)

## Running Locally

Simply open `index.html` in any modern web browser. No build process or dependencies required!

## Technologies Used

- HTML5
- CSS3 (with responsive design)
- Vanilla JavaScript

## Game Controls

- **Click** on a piece to select it for your opponent
- **Click** on an empty board space to place your piece
- **New Game** button to reset and start over

Enjoy playing Quarto!
