# Cannon Strike

A mobile-optimized 3D cannon shooting game built with Three.js and physics simulation.

## Features

- **3D Graphics**: Built with Three.js for smooth 3D rendering
- **Realistic Physics**: Uses cannon-es physics engine for authentic projectile motion
- **Mobile Optimized**: Touch controls designed for mobile devices
- **Interactive Gameplay**:
  - Drag to aim the cannon
  - Hold the FIRE button to charge power
  - Hit targets to score points

## Controls

- **Aim**: Drag anywhere on the screen to rotate the cannon left/right
- **Fire**: Hold the red FIRE button to charge power, release to shoot
- **Score**: Land cannonballs in the holes to earn 100 points each

## Technical Details

- Pure HTML/CSS/JavaScript
- Three.js for 3D rendering
- cannon-es for physics simulation
- No build process required
- Fully responsive design
