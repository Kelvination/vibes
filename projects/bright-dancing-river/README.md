# F1 Pulse

A static, single-page Formula 1 info site. Includes driver and constructor cards, a selection of calendar rounds, key historical milestones, and fast facts.

## Files
- `index.html` — page structure
- `style.css` — F1-inspired dark theme with team accents
- `app.js` — driver/team/calendar data and rendering

## Run locally
```bash
python -m http.server 8000
```
Then open http://localhost:8000.

Data is illustrative and intended for demo purposes.
